LocaVoiture - Car Rental Management for Easy Vehicle, User, and Reservation Control.

This application provides a comprehensive platform for managing car rental operations, allowing users to browse vehicles and make reservations with ease.

Core Features of PowerGym:

  • User Management: Each user can create an account, including personal details such as their name, email, phone number, and password.
  • Vehicle Management: The app allows administrators to manage the vehicle lineup by adding new car models, updating vehicle details, and organizing cars into different categories. This helps users easily find the type of vehicle they need for their rental.
  • Reservation: Users can enter the total number of rental days, select the pick-up and drop-off locations, and specify the start date for their reservation.
  • Admin Dashboard: A powerful admin interface enables administrators to manage vehicle listings, categories, and user profiles. Admins can view all customer bookings.

Technologies Used:

Frontend

  • HTML, CSS and Tailwind CSS for responsive and modern design.

Backend

  • PHP (without a framework) for robust server-side logic, handling user accounts, car listings, and reservations.

Database

  • MySQL for reliable data storage, managing user information, vehicle details, and reservations.
